# 金银 Planet 公司老板 - 金银 形象设计

## 角色设定

**原型：** 美剧《破产姐妹》中的 Max Black

**核心特征：**
- 金发（长发，带霓虹蓝/金色挑染）
- 蓝色眼睛（带科技感霓虹光）
- 微胖身材、曲线美、性感
- 表情：搞笑、自信、毒舌但暖心（一边挑眉一边坏笑）
- 经典姿势：一手叉腰，一手拿酒杯

**服装：**
- 未来感赛博朋克风格
- 黑色紧身衣 + 金色镶边
- 霓虹灯带装饰（蓝色/金色）
- 佩戴"AI 星球老板"工牌

**背景：**
- 赛博朋克酒吧氛围
- 霓虹灯牌"金银 Planet"
- 暖色调灯光（橙色、黄色）
- 吧台、砖墙元素

**风格要求：**
- 漫威漫画风 + 赛博朋克
- 活泼、搞笑、不严肃
- 色彩鲜明：黑/金/霓虹蓝/暖橙

## 推荐使用工具

1. **Midjourney** - 最佳质量，适合漫画风格
2. **DALL-E 3** - 通过 Bing Image Creator 免费使用
3. **Stable Diffusion XL** - 开源，可本地部署
4. **Leonardo.ai** - 免费额度，适合角色设计

## 使用建议

- 使用 `--ar 16:9` 或 `--ar 3:4` 获得更好的构图
- 添加 `--stylize 750` 增强艺术效果（Midjourney）
- 如需要更多细节，可添加 `high detail, 8k, masterpiece`
- 赛博朋克风格可添加 `cyberpunk 2077 style, neon noir`
