# 🎬 王家卫风格 PPT 封面设计

## 设计理念

**王家卫电影美学核心：**
- 浓郁高对比色调
- 冷暖对比（蓝绿 vs 橙黄）
- 模糊/拖影/散景光斑
- 雨景/玻璃反光
- 霓虹灯元素
- 时间流逝感
- 孤独但浪漫的都市氛围

---

## 📽️ 方案一：重庆森林 (Chongqing Express)

**色调：** 蓝绿色调为主，冷色系
**灵感：** 《重庆森林》经典绿调

### 视觉元素

| 元素 | 实现方式 |
|------|----------|
| 模糊霓虹光斑 | 多层径向渐变 + 高斯模糊 |
| 散景光斑 | 动画浮动光点层 |
| 雨景效果 | CSS 动画线性渐变 |
| 玻璃反光 | 斜向半透明渐变 |
| 暗角 | 椭圆径向渐变 |
| 胶片颗粒 | SVG 噪点滤镜 |

### 人物处理

```css
filter: 
    contrast(1.3)      /* 高对比 */
    saturate(0.7)      /* 低饱和 */
    hue-rotate(-10deg) /* 冷色调偏移 */
    brightness(0.9)    /* 略暗，半剪影 */
    drop-shadow(...)   /* 暖色轮廓光 */
```

### 文字设计

- **标题：** 竖排"金银 Planet"，书法体（Ma Shan Zheng）
- **时间戳：** "2026 年 3 月 16 日，下午 3 点之前的一分钟"
- **诗意文字：** "一个人，22 个 AI，一个星球。在重庆的夜里，我们都是孤独的行星。"

### 氛围

> 冷色调都市夜景，孤独但不悲伤，适合知识分享、投资分析场景

**文件：** `cover-wong-01-chongqing.png` (2.5MB)

## 📽️ 方案二：花样年华 (In the Mood for Wealth)

**色调：** 暖黄色调为主，怀旧感
**灵感：** 《花样年华》经典暖黄

### 视觉元素

| 元素 | 实现方式 |
|------|----------|
| 复古霓虹灯 | 大号书法字 + 多层发光 |
| 暖色光斑 | 橙黄色径向渐变 |
| 烟雾/朦胧感 | 大范围模糊渐变 |
| 散景尘埃 | 动画浮动微粒 |
| 时钟元素 | CSS 绘制时钟装饰 |
| 镜子反射 | 底部倒影效果 |

### 人物处理

```css
filter: 
    contrast(1.25)     /* 高对比 */
    saturate(0.65)     /* 低饱和 */
    sepia(0.3)         /* 复古棕调 */
    hue-rotate(-5deg)  /* 暖色调偏移 */
    brightness(0.85)   /* 略暗 */
    drop-shadow(...)   /* 暖色光晕 */
```

### 文字设计

- **标题：** 竖排"金银 Planet"，书法体，强烈发光
- **副标题：** "In the Mood for Wealth"（竖排）
- **诗意文字：** "如果记忆是一个星球，我希望它永远不要熄灭。"
- **时间戳：** "二零二六年 · 春 · 重庆"

### 氛围

> 暖色调怀旧感，浪漫但有距离感，适合年度总结、发布会场景

**文件：** `cover-wong-02-inthemood.png` (2.5MB)

## 🎨 技术实现详解

### 1. 竖排文字

```css
writing-mode: vertical-rl;
text-orientation: mixed;
```

### 2. 书法字体

```css
@import url('https://fonts.googleapis.com/css2?family=Ma+Shan+Zheng&display=swap');
font-family: 'Ma Shan Zheng', cursive;
```

### 3. 散景光斑

```css
background-image: 
    radial-gradient(8px 8px at 15% 25%, rgba(255,255,255,0.8) 0%, transparent 50%),
    radial-gradient(6px 6px at 25% 35%, rgba(255,200,150,0.6) 0%, transparent 50%);
filter: blur(3px);
```

### 4. 王家卫色调

**冷调（重庆森林）：**
- 背景：`#0a1a1a` → `#2a5a5a`（蓝绿渐变）
- 光斑：粉色、青色、橙色对比

**暖调（花样年华）：**
- 背景：`#1a1510` → `#3a2a18`（棕黄渐变）
- 光斑：橙色、金色、琥珀色

### 5. 电影感细节

| 效果 | 作用 |
|------|------|
| 胶片颗粒 | 增加复古质感 |
| 暗角 | 聚焦视线中心 |
| 玻璃反光 | 模拟透过玻璃拍摄 |
| 雨景动画 | 动态氛围感 |
| 时钟元素 | 时间主题呼应 |
