## 📊 筹码集中度分析 (`chip_concentration.csv`)

### 核心指标

| 指标 | 计算公式 | 阈值参考 |
|------|---------|---------|
| 股东户数变化率 | (本期 - 上期)/上期 × 100% | <-5% 集中度提升 |
| 人均持股数 | 总股本/股东户数 | 上升=集中度提升 |
| 筹码集中度 | 前 10 大流通股东占比 | >50% 高度集中 |

### 数据源优先级

1. 同花顺 iFind (最准确)
2. AKShare (免费，可能有延迟)
3. 公司定期报告 (最权威，但滞后)

## ⚠️ 风控检查表 (`risk_checklist.csv`)

### 检查项

- [ ] 单只股票仓位是否超过 20%?
- [ ] 整体仓位是否在目标范围内?
- [ ] 是否有股票触及止损线?
- [ ] 股东户数是否异常变化 (>10%)?
- [ ] 是否有重大利空消息?

## 📝 使用规范

### 时间戳格式

```
YYYY-MM-DD HH:MM  (例: 2026-03-14 14:10)
```

### 增量更新规则

1. **不要覆盖历史数据** - 每次更新新增一行
2. **保留原始文件** - 复制新版本时用 `filename_v2.csv` 格式
3. **变更记录** - 在文件末尾添加更新日志

### 数据验证

- 关键数据必须 2 个以上来源交叉验证
- 价格数据标注来源和时间
- 股东户数标注报告期
