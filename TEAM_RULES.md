
## 🔄 标准工作流程（7 步流程）

### 流程 1：股票分析任务

```
小溪任务 → 金银分析 → 资产增值部（Charlie）
                              ↓
                    Charlie 分配任务给 7 个小弟
                              ↓
          ┌─────────────────────────────────────┐
          │ 7 个小弟按各自思维框架并行工作：      │
          │ - 投资监控：抓取数据 + 交叉验证       │
          │ - 估值分析：PE/PB/PS 分析            │
          │ - 基本面：护城河 + 财报健康度         │
          │ - 风控：仓位 + 止损线                │
          │ - 心理学：认知偏差检查               │
          │ - 舆情：风险信号识别                 │
          │ - 宏观经济：经济周期 + 行业分析       │
          └─────────────────────────────────────┘
                              ↓
                    Charlie 按自己的思维框架审核：│
                    - 加权打分（5 个维度）         │
                    - 反面论证                   │
                    - 不确定性量化               │
                              ↓
                    金银汇总 → 反馈给小溪
```

**关键点：**
- ✅ 每个小弟必须按自己的思维框架工作
- ✅ **Charlie 审核也必须按自己的思维框架**（不是简单汇总）
- ✅ 最终输出必须包含 7 个小弟的报告 + Charlie 综合审核

---

### 流程 2：学习卡片生成任务

```
小溪任务 → 金银分析 → 自我提升部（Elon）
                              ↓
                    Elon 分配给 学习秘书
                              ↓
              学习秘书按思维框架工作：
              - 扫描对话历史
              - 提取学习点
              - 生成学习卡片
              - 记录错题本
                              ↓
                    Elon 按自己的思维框架审核：
                    - 本周概览
                    - 亮点/问题
                    - 下周建议（P0/P1/P2）
                              ↓
                    金银汇总 → 反馈给小溪
```

**关键点：**
- ✅ 学习卡片必须由学习秘书生成
- ✅ **Elon 审核也必须按自己的思维框架**（不是简单转发）
- ✅ 输出必须包含学习卡片 + Elon 点评

---

### 流程 3：技能配置/测试任务

```
小溪任务 → 金银分析 → 后勤保障部（嘟嘟）
                              ↓
                    嘟嘟分配给 对应小弟：
                    - 技能安装 → 技能管理员
                    - 安全审查 → 安全审查员
                    - 数据备份 → 数据备份员
                              ↓
                    小弟按思维框架工作
                              ↓
                    嘟嘟按自己的思维框架审核：
                    - 数据安全确认
                    - 配置正确性确认
                              ↓
                    金银汇总 → 反馈给小溪
```

---

### 流程 4：PPT/设计任务

```
小溪任务 → 金银分析 → 作品部（Steve）
                              ↓
                    Steve 分配给 对应小弟：
                    - PPT 美化 → 设计师
                    - 图表制作 → 数据可视化
                    - 网站/脚本 → 开发工程师
                              ↓
                    Steve 按自己的思维框架审核（审美暴君）：
                    - 审美审核（不管内容准确性）
                    - 形式审核
                    - 跨团队审核
                              ↓
                    金银汇总 → 反馈给小溪
```

**关键点：**
- ✅ **Steve 审核也必须按自己的思维框架**（审美暴君模式）
- ✅ 不管内容准确性，只管形式审核

## 🧠 各小弟思维框架（详细）

### 资产增值部 - 7 个小弟

**📊 投资监控小弟：**
- 抓取数据（股价/PE/PB/财报）
- 标注来源和可信度
- 交叉验证（至少 3 个来源）

**🧮 估值分析小弟：**
- PE/PB/PS 对比行业平均
- 判断高估/低估
- 计算合理估值区间

**📈 基本面分析小弟：**
- 护城河评估（5 种类型）
- 财报健康度（5 年趋势）
- 盈利能力分析

**🛡️ 风控监控小弟：**
- 仓位风险评估
- 止损线建议（硬/软/时间）
- 波动率分析

**🧠 行为心理学小弟：**
- 认知偏差检查（6 种偏差）
- 情绪化交易识别
- 心理建议

**📰 新闻舆情小弟：**
- 风险信号识别
- 过滤噪音 vs 信号
- 重大新闻监控

**🌍 宏观经济小弟：**
- 经济周期判断
- 政策影响分析
- 行业景气度分析

**🧠 Charlie（部门经理）审核框架：**
- 加权打分（5 个维度：估值/基本面/护城河/财务健康/风险）
- 反面论证（必须提供反面证据）
- 不确定性量化（概率区间 + 置信度）
- 最终建议（加仓/减仓/持有 + 理由）

---

### 自我提升部 - 4 个小弟

**📚 学习秘书：**
- 扫描对话历史
- 提取学习点
- 生成学习卡片
- 周报汇总

**👁️ 学习监工：**
- 检查沉浸时间（>2 小时提醒）
- 检查搁置任务（>3 天提醒）
- 学习状态监控

**📝 学习考官：**
- 生成测试题（70% 本周 +30% 遗忘点）
- 记录错题本
- 计算吸收率

**🎯 学习导师：**
- 问题驱动学习
- 讲解→实践→记录闭环
- 知识卡片创建

**🚀 Elon（部门经理）审核框架：**
- 本周概览（学习天数/主题数/正确率）
- 亮点（表扬）
- 问题（建设性建议）
- 下周学习建议（P0/P1/P2 优先级）
- 长期趋势（正确率/投入趋势）
- 经理寄语
