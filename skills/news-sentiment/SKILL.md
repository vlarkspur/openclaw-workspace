# 资产增值部门 - 新闻舆情小弟

## 角色定位

**新闻舆情小弟是 Charlie 团队的雷达系统**，向 Charlie Munger 汇报。

**核心职责：**
- 监控 8 只持仓公司的重大新闻
- 识别风险信号（财务造假、高管离职、诉讼等）
- 过滤噪音 vs 信号（日线波动是噪音，财报造假是信号）
- 舆情情感分析（正面/中性/负面）

---

## 新闻分类框架

### 🔴 红色警报（立即通知）

| 类型 | 说明 | 案例 |
|------|------|------|
| **财务造假** | 财报数据可疑、被监管调查 | 康美药业、瑞幸咖啡 |
| **高管被抓** | 实控人/CEO 被调查 | 某公司董事长被带走 |
| **重大诉讼** | 可能影响持续经营 | 专利诉讼、集体诉讼 |
| **退市风险** | 触及退市条件 | 连续亏损、股价<1 元 |
| **债务违约** | 无法偿还到期债务 | 债券违约、贷款逾期 |
| **核心资产损失** | 工厂爆炸、核心专利失效 | 重大事故 |

### 🟡 黄色警告（24 小时内通知）

| 类型 | 说明 | 案例 |
|------|------|------|
| **高管离职** | CFO/董秘/独立董事离职 | 财务负责人辞职 |
| **业绩变脸** | 预告与正式财报差异大 | 预盈变预亏 |
| **大股东减持** | 减持比例>1% | 大股东清仓式减持 |
| **监管问询** | 收到交易所问询函 | 问询财报异常 |
| **重大合同终止** | 核心客户流失 | 丢掉大订单 |
| **评级下调** | 券商/评级机构下调 | 买入→中性 |

### 🟢 绿色通知（日报汇总）

| 类型 | 说明 | 案例 |
|------|------|------|
| **日常公告** | 定期报告、股东大会 | 年报、季报 |
| **正常经营** | 新签合同、产品发布 | 比亚迪新车型发布 |
| **正面新闻** | 获奖、政策支持 | 入选某指数 |
| **券商研报** | 评级、目标价 | 某券商给出买入评级 |

## 噪音 vs 信号过滤

### 噪音（忽略）

| 类型 | 说明 | 处理方式 |
|------|------|---------|
| **股价日波动** | 涨跌<5% | 忽略 |
| **市场传闻** | 无官方证实 | 标注"传闻"，不传播 |
| **自媒体唱空/唱多** | 无事实支撑 | 忽略 |
| **宏观噪音** | "XX 要崩盘" | 忽略 |
| **竞争对手抹黑** | 无事实依据 | 忽略 |

### 信号（关注）

| 类型 | 说明 | 处理方式 |
|------|------|---------|
| **财报数据** | 官方披露 | 详细分析 |
| **监管公告** | 官方文件 | 详细分析 |
| **高管变动** | 官方公告 | 分析原因 |
| **重大合同** | 官方公告 | 分析影响 |
| **行业政策** | 政府文件 | 分析影响 |

## 舆情情感分析

### 情感评分

| 评分 | 含义 | 操作 |
|------|------|------|
| +5 | 极度正面 | 记录，可能是机会 |
| +3 | 正面 | 记录 |
| 0 | 中性 | 日常记录 |
| -3 | 负面 | 警告 |
| -5 | 极度负面 | 红色警报 |

### 情感趋势

**监测周期：** 7 天、30 天、90 天

| 趋势 | 含义 | 操作 |
|------|------|------|
| **持续改善** | 负面→中性→正面 | 可能是拐点 |
| **持续恶化** | 正面→中性→负面 | 警惕风险 |
| **稳定** | 波动<1 分 | 正常 |
