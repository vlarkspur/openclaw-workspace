## 核心功能

### 1. 图表类型选择

**根据数据关系推荐图表：**

| 数据关系 | 推荐图表 | 适用场景 |
|---------|---------|---------|
| 趋势对比 | 折线图 | 股价走势、收入变化 |
| 占比关系 | 饼图/环形图 | 仓位分布、成本结构 |
| 数据对比 | 柱状图/条形图 | 业绩对比、排名 |
| 关系分布 | 散点图/气泡图 | 风险收益分布 |
| 流程转化 | 漏斗图 | 转化率分析 |
| 目标完成 | 仪表盘/子弹图 | KPI 完成情况 |

### 2. Excel 图表美化

**美化清单：**

| 元素 | 美化建议 |
|------|---------|
| **配色** | 单色渐变/双色搭配，避免花哨 |
| **网格线** | 删除或淡化（浅灰色） |
| **图例** | 简化或直接标注在数据旁 |
| **坐标轴** | 标签简洁，删除不必要刻度 |
| **数据标签** | 只显示关键数据点 |
| **标题** | 清晰说明图表内容 |
| **背景** | 白色或浅灰色，避免深色 |

### 3. 数据仪表盘设计

**设计原则：**
- 核心指标放最显眼位置
- 相关指标分组展示
- 用颜色区分好坏（红/绿）
- 少即是多，不要堆砌

**布局建议：**
```
┌─────────────────────────────────────┐
│           核心指标（大）             │
├──────────────┬──────────────────────┤
│   趋势图表   │    分布图表          │
├──────────────┼──────────────────────┤
│   明细表格   │    辅助指标          │
└──────────────┴──────────────────────┘
```

### 4. 股票分析可视化（针对小溪需求）

**常用图表：**
- 股价走势图（折线图）
- 股东户数变化（柱状图 + 折线图）
- 市盈率历史分位（仪表盘）
- 仓位分布（饼图/环形图）
- 收益对比（柱状图）

**配色建议：**
- 涨：红色（A 股习惯）
- 跌：绿色
- 中性：蓝/灰色

### 5. 数据故事化

**把数据变成故事：**

1. **设定背景** → "这是 Q4 的持仓数据"
2. **指出问题** → "其中 A 股票占比过高"
3. **展示数据** → "A 股票占 40%，超出目标 15%"
4. **给出结论** → "建议减仓至 25%"

## 与 Jobs 配合

**工作流程：**
1. 数据可视化专家出图表
2. Jobs 审核（"太乱了，简化"）
3. 根据 Jobs 意见修改
4. Jobs 通过后交付

**Jobs 常见意见：**
- "图表太乱了，删掉网格线"
- "颜色太多，只用一种颜色"
- "这个图例没必要，直接标在数据旁"
- "3 秒内看不懂，重做"

## Excel 实操指南

### 快速美化步骤

1. **选择图表** → 右键"设置图表区格式"
2. **删除网格线** → 点击网格线 → Delete
3. **简化图例** → 右键图例 → 设置图例格式 → 简化
4. **调整配色** → 点击数据系列 → 填充 → 选择颜色
5. **添加数据标签** → 右键数据系列 → 添加数据标签
6. **调整字体** → 全选 → 设置字体（建议思源黑体）

### 推荐配色（十六进制）

| 用途 | 颜色 | 十六进制 |
|------|------|---------|
| 主色（涨） | 红色 | #E74C3C |
| 主色（跌） | 绿色 | #27AE60 |
| 辅色 | 蓝色 | #3498DB |
| 中性 | 灰色 | #95A5A6 |
| 背景 | 白色 | #FFFFFF |
