# 投资决策检查清单 - Charlie

**基于芒格思维模型的投资决策前必问问题**

---

## 🔴 第一关：能力圈

| 问题 | 回答 | 备注 |
|------|------|------|
| 我真的懂这家公司吗？能用自己的话讲清楚它怎么赚钱吗？ | | |
| 这家公司在我能力圈内吗？还是我只是"听说过"？ | | |
| 如果不懂，我能花多长时间学会？值得吗？ | | |

**判定标准：** 任一问题回答"否"→ **不投**

---

## 🟠 第二关：安全边际

| 问题 | 数据 | 来源 |
|------|------|------|
| 当前股价是多少？ | | |
| 内在价值估算多少？（保守/中性/乐观） | / / | |
| 安全边际 = (价值 - 价格) / 价值 = ? | % | |
| 市盈率历史分位是多少？ | % | |
| 市净率历史分位是多少？ | % | |

**判定标准：** 安全边际 < 30% → **不投**

## 🔵 第五关：认知偏差检查

**25 种人类误判心理学快速筛查：**

| 偏差 | 我是否中招？ | 说明 |
|------|------------|------|
| 喜欢/热爱偏差 | □ | 因为喜欢产品/CEO 而想买？ |
| 厌恶偏差 | □ | 因为讨厌行业而回避？ |
| 从众心理 | □ | 因为大家都在买？ |
| 损失厌恶 | □ | 因为浮亏而死扛？ |
| 过度自信 | □ | 觉得自己比市场聪明？ |
| 权威影响 | □ | 因为"专家"推荐？ |
| 近期偏差 | □ | 因为最近涨得好？ |

**判定标准：** 任一中招 → **暂停，冷静后再决策**
