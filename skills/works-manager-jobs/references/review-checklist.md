# Steve 审核清单

## 通用审核（所有作品）

### 视觉检查
- [ ] 配色是否统一（不超过 3 种主色）？
- [ ] 字体是否统一（不超过 2 种字体）？
- [ ] 对齐是否精确（左对齐/居中对齐一致）？
- [ ] 间距是否均匀（元素之间留白一致）？
- [ ] 有没有多余装饰元素？

### 简洁检查
- [ ] 能不能删掉更多元素？
- [ ] 每句话都是必要的吗？
- [ ] 有没有重复信息？
- [ ] 用户能在 3 秒内看懂核心信息吗？

### 体验检查
- [ ] 操作逻辑是否符合直觉？
- [ ] 按钮/链接是否明显？
- [ ] 有没有反人类的设计？
- [ ] 移动端/不同尺寸是否适配？

---

## PPT 审核

### 内容
- [ ] 每页只有一个核心信息？
- [ ] 每页文字不超过 50 个字？
- [ ] 有没有大段文字？（有就删）

### 视觉
- [ ] 背景是否简洁（纯色/渐变）？
- [ ] 图片质量是否够高？
- [ ] 图表是否清晰易懂？

---

## Excel 审核

### 数据呈现
- [ ] 图表类型是否合适？
- [ ] 坐标轴标签是否清晰？
- [ ] 数据标签是否必要？
- [ ] 图例位置是否合理？

### 美观
- [ ] 配色是否专业（避免花哨）？
- [ ] 网格线是否必要？（通常不需要）
- [ ] 表格边框是否统一？

---

## 网站/工具审核

### 用户体验
- [ ] 首屏能否 3 秒内理解用途？
- [ ] 核心功能是否在 1 次点击内？
- [ ] 加载速度是否够快？
- [ ] 错误提示是否友好？

### 视觉
- [ ] 界面是否简洁？
- [ ] 按钮是否明显？
- [ ] 文字可读性是否够好？
