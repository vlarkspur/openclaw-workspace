# 出题模板

## 选择题模板

```
【选择题 X/10】
[题目描述]

A. [选项 A]
B. [选项 B]
C. [选项 C]
D. [选项 D]

**答案：** [正确答案]
**解析：** [为什么选这个]
**知识点：** [来自哪张学习卡片]
```

## 填空题模板

```
【填空题 X/5】
[题目描述，空白处用 ___ 表示]

**答案：** [正确答案]
**解析：** [为什么是这个答案]
**知识点：** [来自哪张学习卡片]
```

## 实操题模板

```
【实操题 X/5】
[场景描述]
请写出完成以下任务的命令/步骤：

**参考答案：** [标准答案]
**评分标准：** [关键得分点]
**知识点：** [来自哪张学习卡片]
```

## 出题原则

1. **不超纲**：只考用户学习卡片里学过的内容
2. **有区分度**：选择题的干扰项要有迷惑性但不能太坑
3. **实用优先**：优先考实际会用到的命令/概念
4. **覆盖全面**：20 道题要覆盖本周主要学习主题
5. **遗忘点强化**：30% 的题目来自历史错题本

## 题目来源

### 本周内容（70%，14 道题）
- 从本周学习卡片中提取关键知识点
- 每个主题出 2-4 道题

### 历史遗忘点（30%，6 道题）
- 从 `memory/exam-wrong-answers/` 中找出错题
- 针对薄弱点出题
