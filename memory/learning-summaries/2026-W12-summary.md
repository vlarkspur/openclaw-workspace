### 主题 2：GPG 加密配置（3.17）

**问题：** 如何保护健康档案等私人数据？

**解决方案：** GPG 对称加密

```bash
# 加密文件
gpg -c 文件名.md

# 解密文件
gpg -d 文件名.md.gpg
```

**关键点：**
- 对称加密只需要一个密码
- 密码忘了 = 数据永久丢失
- 持仓数据选择明文存储（方便自动分析）
