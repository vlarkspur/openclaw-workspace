# 持仓股票分析框架（Charlie 方法）

**创建日期：** 2026-03-20  
**最后更新：** 2026-03-20 11:09  
**适用范围：** 所有持仓股票调仓分析  
**方法论：** 从大到小四层过滤 + 反过来想反面论证

---

## 一、核心原则

> **第一句：** 从大到小，四层过滤（国际→国家→行业→公司）  
> **第二句：** 反过来想，永远质疑（每个结论必须有反面论证）

**芒格原话：** "反过来想，总是反过来想。"（Invert, always invert.）

---

## 二、分析流程（标准 SOP）

### 步骤 1：数据收集（30 分钟）
- [ ] 财务数据（同花顺/富途）
- [ ] 股价数据（腾讯财经）
- [ ] 行业数据（研报/新闻）
- [ ] 宏观数据（国家统计局）
- [ ] **年报数据（如已发布）**

### 步骤 2：四层过滤（60 分钟）

#### 🌍 第一层：国际局势（地缘政治）
- [ ] 全球主要经济体关系（合作/对抗）
- [ ] 供应链是否依赖冲突地区
- [ ] 关键资源供应安全（芯片/能源/稀有金属）
- [ ] 关税/贸易壁垒风险

**2026 年关键背景：**
- 美以伊冲突（卡塔尔 LNG 设施遭袭，年损失 200 亿美元，修复 5 年）
- 霍尔木兹海峡安全（六国声明保障，伊朗拟征通行费）
- 芯片危机（DRAM Q1 +90-95%，或持续至 2030 年）

#### 🇨🇳 第二层：国家政策（宏观周期）
- [ ] 当前经济周期判断
- [ ] GDP 目标 vs 实际增速
- [ ] CPI/PPI 趋势
- [ ] PMI 扩张/收缩
- [ ] 房地产影响
- [ ] 出口/基建/消费增速
- [ ] 货币政策（降准降息预期）
- [ ] 产业政策（以旧换新/设备更新）

**2026 年关键数据：**
| 指标 | 数值 | 含义 |
|------|------|------|
| GDP 目标 | 4.5-5% | 30 年来首次低于 5% |
| 2025 年实际 GDP | +5.0% | 完成目标 |
| CPI 目标 | 2% 左右 | 通缩压力 |
| 2025 年实际 CPI | 0% | 与上年持平 |
| 2026 年 2 月 CPI | +1.3% | 温和回升 |
| PMI（2026 年 2 月） | 49.0% | 收缩区间 |
| 出口 | +18.3% | 外需强 |
| 房地产投资 | -11.1% | 拖累 |
| 基建投资 | +11.4% | 政策发力 |
| 美联储利率 | 5.25-5.5% | 连续 6 次不变 |
| 美联储 2026 预期 | 降息 2-3 次 | 下半年宽松 |
| 人民币汇率 | 6.7-7.2 | 双向波动 |
| A 股估值（沪深 300） | PE 14.2 倍，66-92% 分位 | 偏高 |
| 港股估值（恒生指数） | PE 12.33 倍，80% 分位 | 偏高 |
| 港股估值（恒生科技） | PE 21.21 倍，16% 分位 | **极度低估** |

#### 🏭 第三层：行业分析（景气度 + 竞争格局）
- [ ] 行业周期（导入/成长/成熟/衰退）
- [ ] 行业增速 vs GDP 增速
- [ ] 竞争格局（寡头/分散）
- [ ] 进入壁垒
- [ ] 替代品威胁
- [ ] 上下游议价能力

**电动车行业关键数据（2026 年）：**
| 公司 | 2025 年销量 | 同比 | 2026 年目标 |
|------|-----------|------|-----------|
| 比亚迪 | 460.24 万辆 | +28% | - |
| 特斯拉 | 163.6 万辆 | -8.6% | - |
| 理想汽车 | 40.63 万辆 | -18.8% | - |
| 小米汽车 | 41 万辆 | 新进入者 | 55 万辆 |

**手机行业关键数据（2026 年）：**
| 指标 | 数值 | 影响 |
|------|------|------|
| 全球手机出货预测 | -2.1% | 收缩 |
| 小米出货预测下调 | -20%+ | 严重收缩 |
| DRAM 价格 Q1 | +90-95% | 成本暴涨 |
| NAND 价格 Q1 | +55-60% | 成本暴涨 |

**行业政策：**
- 以旧换新（报废）：新能源车 12% 最高 2 万，燃油车 10% 最高 1.5 万
- 以旧换新（置换）：新能源车 8% 最高 1.5 万，燃油车 6% 最高 1.3 万

#### 🏢 第四层：公司分析（基本面 + 估值）

**财务健康（6 项核心指标）：**
- [ ] 营收增速（vs 行业平均）
- [ ] 归母净利润增速
- [ ] 毛利率趋势
- [ ] 净利率趋势
- [ ] ROE（>15% 为优）
- [ ] 经营现金流（是否为正）
- [ ] 资产负债率（<60% 为优）

**护城河（5 种类型）：**
- [ ] 无形资产（品牌/专利/牌照）
- [ ] 成本优势（规模效应/独特工艺）
- [ ] 网络效应
- [ ] 转换成本
- [ ] 有效规模（利基市场垄断）

**估值（4 种方法）：**
- [ ] PE-TTM（vs 历史分位）
- [ ] PB（vs 行业平均）
- [ ] PEG（PE/增速，<1 为优）
- [ ] DCF（内在价值估算）

**催化剂：**
- [ ] 新产品发布
- [ ] 新市场开拓
- [ ] 政策利好
- [ ] 财报超预期

---

### 步骤 3：反面论证（30 分钟）

**每个结论必须回答 5 个反面问题：**
1. 如果经济衰退，公司能活吗？
2. 如果行业增速减半，还能增长吗？
3. 如果龙头降价 30%，能跟进吗？
4. 如果技术被颠覆，有备选吗？
5. 如果 PE 再跌 50%，逻辑还成立吗？

**认知偏差检查（芒格 25 种误判）：**
- [ ] 喜欢/热爱倾向（是否因为喜欢品牌而忽视风险）
- [ ] 避免不一致性倾向（是否因为已持仓而倾向加仓）
- [ ] 社会认同倾向（是否因为"别人都买"而跟风）
- [ ] 稀缺性错误倾向（是否被"错过就没机会"忽悠）
- [ ] 权威错误倾向（是否因为"巴菲特说"而盲从）

---

### 步骤 4：评分决策（10 分钟）

**决策矩阵（满分 100 分）：**

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 国际局势 | 15% | 无风险 15 分，中等风险 8 分，高风险 0 分 |
| 国家政策 | 15% | 直接受益 15 分，中性 8 分，被拖累 0 分 |
| 行业景气 | 20% | 高成长 20 分，成熟 12 分，衰退 0 分 |
| 公司基本面 | 25% | 财务健康 + 护城河宽 25 分，中等 15 分，差 0 分 |
| 估值 | 15% | 显著低估 15 分，合理 8 分，高估 0 分 |
| 催化剂 | 10% | 明确催化剂 10 分，模糊 5 分，无 0 分 |

**决策阈值：**

| 总分 | 决策 | 仓位 |
|------|------|------|
| 80-100 | 🟢 强烈买入 | 10-15% |
| 60-79 | 🟢 加仓 | 5-10% |
| 40-59 | 🟡 持有/观望 | 维持现有 |
| 20-39 | 🟡 减仓 | 降至 3-5% |
| 0-19 | 🔴 清仓 | 0% |

---

### 步骤 5：撰写报告（30 分钟）

**报告结构：**
1. 核心结论（先说答案）
2. 四层分析（从大到小）
3. 反面论证（反过来想）
4. 风控建议（仓位/止损）
5. 催化剂/观察点

**总耗时：** 约 2.5 小时/只股票

## 五、数据源优先级

| 数据类型 | 优先来源 | 备选来源 | 验证要求 |
|---------|---------|---------|---------|
| 宏观数据 | 国家统计局 | 政府工作报告 | 2 个来源交叉验证 |
| 行业数据 | Counterpoint/IDC | 行业协会 | 2 个来源交叉验证 |
| 财务数据 | 同花顺/富途 | 东方财富 | 以年报为准 |
| 股价数据 | 腾讯财经 | 东方财富 | 实时 |
| 新闻舆情 | 百度/Google | 财联社/华尔街见闻 | 多源验证 |

**关键数据定义（必须 2-3 个来源交叉验证）：**
- 直接影响买卖决策（股东户数、归母净利润、融资余额）
- 触发风控阈值
- 涉及金额 > 1 万元或仓位 > 5%

## 七、关键教训（从错误中学习）

### [LRN-20260319-001] 小米分析漏掉 3 个关键背景
- **问题：** 初步判断"加仓"，漏掉芯片危机、地缘政治、宏观周期
- **纠正：** 加入三层背景后改为"暂缓加仓"
- **教训：** 不要套用教科书，数据比理论重要

### [LRN-20260319-002] 数据源优先级错误
- **问题：** 腾讯财经 PE 数据错误（37.17 vs 实际 20.33，差异 83%）
- **纠正：** 更新数据源优先级：同花顺 F10/富途 > 东方财富 > 腾讯财经
- **教训：** 关键数据必须交叉验证

### [LRN-20260320-003] CPI 数据错误
- **问题：** 文档中 CPI 目标写 0.4-0.8%，实际政府工作报告是 2%
- **纠正：** 更新为 2%，并补充 2025 年实际 0%、2026 年 2 月 +1.3%
- **教训：** 宏观数据必须以官方为准
