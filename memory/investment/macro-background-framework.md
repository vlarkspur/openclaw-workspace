## 🚨 今天漏掉的 3 个关键背景

### 1️⃣ 中国经济周期判断错误

**原判断：** 复苏后期 → 过热初期 ❌

**修正判断：** 弱复苏 + 结构分化 ✅

| 指标 | 原数据 | 修正后数据 | 含义 |
|------|--------|-----------|------|
| GDP 目标 | 5% | **4.5-5%**（30 年来首次低于 5%） | 政府承认潜在增速下降 |
| CPI | 0.7% | **0.4-0.8%** | 通缩压力，非通胀 |
| PMI | 50.8 | **49.0%**（2 月） | 收缩区间 |
| 消费 | +2.8% | +2.8% | 弱复苏 |
| 出口 | +18.3% | +18.3% | 外需强 |
| 房地产投资 | - | **-11.1%** | 拖累 |
| 基建投资 | - | +11.4% | 政策发力 |

**对持仓分析的影响：**
- ✅ 政策宽松持续（降准降息预期）
- ✅ 扩内需政策延续（以旧换新）
- ⚠️ 消费复苏缓慢（手机/汽车换机周期拉长）
- ❌ 房地产拖累（IoT 设备、家电需求受影响）
- ❌ 不是全面牛市（结构分化）

### 3️⃣ 美以伊冲突对半导体影响（之前完全漏掉）

**影响路径：**

| 路径 | 说明 | 状态 |
|------|------|------|
| **氦气供应** | 卡塔尔占全球 30%，设施遭袭关闭 LNG | ⚠️ 风险 |
| **油价上涨** | 推高电力成本，影响晶圆厂运营 | ⚠️ 已发生 |
| **物流链** | 霍尔木兹海峡运输受阻 | ⚠️ 风险 |
| **台积电 ADR** | 已下跌约 10% | ❌ 已反应 |
| **SK 海力士** | 已下跌 14% | ❌ 已反应 |
| **三星电子** | 已下跌 15% | ❌ 已反应 |

**对持仓股票的影响：**
- 间接影响：晶圆厂成本上涨 → 芯片涨价 → 终端成本压力
- 直接影响：有限（持仓公司无晶圆厂）
- 长期风险：若冲突持续至 2026 年下半年，可能引发全球半导体供应链中断

## 📋 后续持仓分析检查清单

**每次调仓分析前，必须回答：**

### 宏观周期检查
- [ ] 当前经济周期是什么？（复苏/过热/滞胀/衰退）
- [ ] GDP 目标与实际增速对比？
- [ ] CPI/PPI趋势？
- [ ] PMI 在扩张还是收缩区间？
- [ ] 房地产对持仓行业的影响？

### 芯片风险检查
- [ ] 持仓公司是否依赖外部芯片供应？
- [ ] 芯片成本占营收比例？
- [ ] 是否有长期供货协议锁定价格？
- [ ] 是否有自研芯片能力？
- [ ] 产品涨价能否传导成本？

### 地缘政治检查
- [ ] 持仓公司供应链是否依赖中东/东亚？
- [ ] 氦气、石油等关键材料供应风险？
- [ ] 物流链是否经过霍尔木兹海峡？
- [ ] 是否有替代供应商？
