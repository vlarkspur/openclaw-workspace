## 二、宏观周期检查（必须回答）

### 2.1 当前经济周期是什么？

**判断：** 弱复苏 + 结构分化

| 指标 | 数值 | 对比亚迪影响 |
|------|------|-------------|
| GDP 目标 | 4.5-5% | 🟡 中性 |
| CPI | 0.4-0.8% | 🟢 利好（通缩环境，成本压力小） |

| 房地产投资 | -11.1% | 🟢 利好（比亚迪不依赖房地产） |
| 消费 | +2.8% | 🟡 中性（汽车消费以旧换新政策对冲） |

| 基建投资 | +11.4% | 🟢 利好（电动公交、出租车更新） |

**结论：** 比亚迪处于**结构性机会**的受益位置（出口 + 政策），不受房地产拖累。

### 2.2 芯片风险检查（小米分析教训）

| 检查项 | 比亚迪情况 | 风险等级 |
|--------|------------|---------|
| 是否依赖外部芯片供应？ | 部分依赖，但自研比例高 | 🟢 低 |
| 芯片成本占营收比例？ | 约 5-8%（低于手机行业 20-30%） | 🟢 低 |
| 是否有长期供货协议？ | 有（与英飞凌、恩智浦等） | 🟢 低 |
| 是否有自研芯片能力？ | **有**（IGBT、MCU、SiC 自研） | 🔴 **重大优势** |
| 产品涨价能否传导成本？ | **能**（电动车定价权强于手机） | 🟢 低 |

**关键差异：**
- 小米：芯片成本占营收 20-30%，无自研能力，无法传导成本
- 比亚迪：芯片成本占营收 5-8%，自研 IGBT/SiC，可传导成本

**结论：** 芯片危机对比亚迪影响**轻微**，反而是竞争优势（对手受冲击更大）。

| 检查项 | 比亚迪情况 | 风险等级 |

**结论：** 地缘政治风险**中等**，但比亚迪全球化布局优于同行。

### 3.2 净利润负增长原因分析

**表面原因：**
- 毛利率下滑 1.57pp（价格战 + 成本上涨）
- 费用率上升（研发投入 + 海外扩张）

**深层原因：**
1. **价格战**：2025 年电动车价格战激烈，比亚迪主动降价保份额
2. **海外扩张成本**：匈牙利、巴西、泰国建厂，前期投入大
3. **研发费用**：2025 年研发投入预计 500 亿+（行业最高）
4. **季节性因素**：Q3 通常是淡季，Q4 冲量

**关键判断：**
- 这是**战略性利润下滑**，不是基本面恶化
- 经营现金流 +408.5 亿，证明盈利能力仍在
- 研发投入是长期竞争力，不是浪费
