## 📋 后续分析框架（必须应用）

### 宏观周期检查
- [ ] 当前经济周期是什么？（弱复苏 + 结构分化）
- [ ] GDP 目标与实际增速对比？（4.5-5%，30 年来首次低于 5%）
- [ ] CPI/PPI趋势？（通缩压力，非通胀）
- [ ] PMI 在扩张还是收缩区间？（49.0%，收缩）
- [ ] 房地产对持仓行业的影响？（-11.1%，拖累）

### 芯片风险检查
- [ ] 持仓公司是否依赖外部芯片供应？
- [ ] 芯片成本占营收比例？
- [ ] 是否有长期供货协议锁定价格？
- [ ] 是否有自研芯片能力？
- [ ] 产品涨价能否传导成本？

### 地缘政治检查
- [ ] 持仓公司供应链是否依赖中东/东亚？
- [ ] 氦气、石油等关键材料供应风险？
- [ ] 物流链是否经过霍尔木兹海峡？
- [ ] 是否有替代供应商？
