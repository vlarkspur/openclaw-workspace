## ✅ 消息落款格式要求

**每条消息都必须加上落款：**

```
💃 金银 Planet · [部门]
[任务类型] 时间：YYYY-MM-DD HH:MM
```

**部门示例：**
- 资产增值部（股票分析、投资建议）
- 后勤保障部（系统配置、工具安装）
- 作品部（PPT、文档、设计）
- 自我提升部（学习、培训）

**任务类型示例：**
- 数据查询时间
- 分析报告时间
- 配置完成时间
- 消息发送时间
- 文件保存时间

**完整示例：**
```
💃 金银 Planet · 资产增值部
数据查询时间：2026-03-20 14:00
```

---

## 💡 为什么需要落款？

1. ✅ **谁做的** —— 金银 Planet + 部门
2. ✅ **什么时候做的** —— 精确到分钟
3. ✅ **便于追溯和复盘** —— 方便后续查找和审计
