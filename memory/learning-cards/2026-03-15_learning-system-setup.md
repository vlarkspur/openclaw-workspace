# 学习系统搭建记录

**日期：** 2026-03-15

**耗时：** 约 30 分钟

---

## 遇到的问题

想建立一个碎片化学习的闭环系统，但不知道应该设多少个 Agent，也不知道知识沉淀应该放在哪里。

---

## 解决方案

### 第一步：确定系统架构

5 个 Agent 闭环：
1. **导师** → 教你（问题驱动）
2. **考官** → 测试你（出题、打分）
3. **复盘** → 分析错误（根因分析）
4. **秘书** → 沉淀知识（整理笔记）
5. **监工** → 盯着你用（定期抽查）

### 第二步：确定知识沉淀位置

- 日常碎片 → `memory/learning-cards/YYYY-MM-DD-topic.md`
- 定期汇总 → 周复盘/月复盘 → Word 文档

### 第三步：确定监工时间

每天 17:30 检查一次：
- 有没有在同一个问题上卡超过 2 小时
- 今天有没有写学习卡片
