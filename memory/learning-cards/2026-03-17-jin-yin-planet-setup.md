## 学习主题 2：GPG 加密配置

### 问题
如何保护健康档案等私人数据？

### 解决方案
使用 GPG 对称加密：

```bash
# 加密文件
gpg -c 文件名.md

# 解密文件
gpg -d 文件名.md.gpg
```

### 关键点
- 对称加密只需要一个密码
- 密码不会显示在屏幕上
- 密码忘了 = 数据永久丢失（无找回功能）
- 持仓数据选择明文存储（方便自动分析）

### 踩过的坑
- 密码不要告诉 AI（会留在对话历史里）
- 加密后记得删除原文件

## 学习主题 4：Handover 文档系统

### 问题
项目中断后如何快速恢复？

### 解决方案
创建 Handover 文档 + 配置自动读取：

```markdown
# 项目 Handover 结构
1. 任务目标
2. 已完成
3. 进行中
4. 问题与解决
5. 关键决策
6. 文件地图
7. 踩坑记录
8. 下一步计划
```

### 自动读取配置
在 Charlie 团队 SKILL.md 中添加：
```markdown
## 🚀 项目启动流程（每次任务前必读）
1. 检查 Handover 文档是否存在
2. 如果存在，先读取了解进度
3. 根据 handover 决定下一步行动
```

### 关键概念
- Handover 是给人/agent 在项目交接时用的
- 同一会话继续 → 不需要读（有会话历史）
- 新会话继续 → 需要读 handover
