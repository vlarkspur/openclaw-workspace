## 一、核心公式汇总

### 1. 安全边际计算
```
安全边际 (%) = 当前股价 / 内在价值 × 100%

Excel 公式：=D2/N2*100
其中：D2=当前股价，N2=内在价值
```

### 2. 估值状态判断
```
Excel 公式：
=IF(O2<60,"深度价值",
  IF(O2<80,"价值区间",
    IF(O2<95,"合理偏低",
      IF(O2<=105,"合理",
        IF(O2<=120,"合理偏高","高估")))))
```

### 3. 投资建议
```
Excel 公式：
=IF(O2<50,"强烈买入",
  IF(O2<60,"买入",
    IF(O2<80,"增持",
      IF(O2<=105,"持有",
        IF(O2<=120,"减持","卖出")))))
```

## 二、内在价值计算方法

### 方法 1：DCF 模型（现金流折现）

**适用：** 比亚迪、理想汽车、小米集团（有稳定现金流的企业）

**公式：**
```
内在价值 = ∑[FCF_t / (1+r)^t] + 终值/(1+r)^n

其中：
- FCF_t = 第 t 年的自由现金流
- r = 折现率 (WACC)
- n = 预测期数 (通常 5 年)
- 终值 = FCF_n × (1+g) / (r-g)
- g = 永续增长率
```

**Excel 实现：**
```excel
年份          0       1       2       3       4       5
自由现金流    -    280     302     327     353     381
折现因子      -    0.913   0.834   0.762   0.696   0.636
现值          -    255.6   252.1   249.0   245.7   242.3

5 年现值合计 = SUM(E4:I4) = 1244.7
终值 = I3*(1+3%)/(9.5%-3%) = 6038.5
终值现值 = 6038.5*0.636 = 3840.5
企业价值 = 1244.7 + 3840.5 = 5085.2 亿
每股价值 = 5085.2/91.17 = 55.77 元
```

**参数说明：**
| 参数 | 比亚迪 | 理想汽车 | 小米集团 |
|------|--------|----------|----------|
| 折现率 (WACC) | 9.5% | 11% | 10% |
| 5 年 FCF 增长率 | 8% | 25% | 18% |
| 永续增长率 | 3% | 4% | 3.5% |

---

### 方法 2：市盈率法 (PE Valuation)

**适用：** 盈利稳定的企业

**公式：**
```
内在价值 = 合理 PE × EPS(TTM)
```

**合理 PE 确定方法：**
```
合理 PE = 行业平均 PE × (1 + 相对增长率/100)

或参考：
- 高增长 (>30%)：30-40 倍
- 中增长 (15-30%)：20-30 倍
- 低增长 (5-15%)：15-20 倍
- 无增长 (<5%)：8-15 倍
```

**Excel 公式：**
```excel
=合理 PE 单元格 * EPS 单元格

示例（比亚迪）：
=27.5 * 3.42 = 94.05 元
```

**各行业合理 PE 参考：**
| 行业 | 合理 PE 区间 | 代表公司 |
|------|-------------|----------|
| 新能源汽车 | 20-35 倍 | 比亚迪、理想 |
| 互联网科技 | 20-35 倍 | 小米、腾讯 |
| 基建 | 6-10 倍 | 中国中铁 |
| 券商/投资 | 10-15 倍 | 吉林敖东 |
| 互联网安全 | 25-45 倍 | 三六零 |
| 娱乐传媒 | 15-25 倍 | 大麦娱乐 |
| 半导体/AI | 30-60 倍 (或 PS 法) | 地平线 |

---

### 方法 3：市净率法 (PB Valuation)

**适用：** 重资产企业、金融企业、周期股

**公式：**
```
内在价值 = 合理 PB × BVPS
```

**合理 PB 确定方法：**
```
合理 PB = ROE / (r - g)

简化参考：
- ROE > 20%：4-6 倍
- ROE 15-20%：3-4 倍
- ROE 10-15%：2-3 倍
- ROE 5-10%：1-2 倍
- ROE < 5%：0.5-1 倍
```

**Excel 公式：**
```excel
=合理 PB 单元格 * BVPS 单元格

示例（中国中铁）：
=0.85 * 9.48 = 8.06 元
```

---

### 方法 4：股息折现法 (DDM)

**适用：** 高分红稳定企业

**公式：**
```
内在价值 = D1 / (r - g)

其中：
- D1 = 下年预期股息 = 当前股息 × (1+g)
- r = 要求回报率
- g = 股息永续增长率
```

**Excel 公式：**
```excel
=(当前股息*(1+ 永续增长率)) / (要求回报率 - 永续增长率)

示例（中国中铁）：
=(0.29*(1+2%)) / (8%-2%) = 0.2958 / 0.06 = 4.93 元
```

---

### 方法 5：市销率法 (PS Valuation)

**适用：** 未盈利但高增长企业

**公式：**
```
合理市值 = 年营收 × 合理 PS
每股价值 = 合理市值 / 总股本
```

**合理 PS 参考：**
| 营收增长率 | 合理 PS 区间 |
|------------|-------------|
| >50% | 15-25 倍 |
| 30-50% | 10-15 倍 |
| 15-30% | 6-10 倍 |
| 5-15% | 3-6 倍 |
| <5% | 1-3 倍 |

## 四、置信度评估方法

### 置信度计算公式
```
置信度 = 基础分 - 风险扣分

基础分：
- 盈利企业：80 分
- 未盈利企业：50 分

风险扣分项：
- 行业竞争激烈：-5 分
- 营收下滑：-10 分
- 连续亏损：-15 分
- 高负债率 (>70%)：-5 分
- 现金流为负：-10 分
- 技术路线不确定：-15 分
- 政策风险：-5 分
```

### 各股票置信度明细

| 股票 | 基础分 | 扣分项 | 最终置信度 |
|------|--------|--------|------------|
| 中国中铁 | 80 | 0 | 85% (向上调整因稳定) |
| 吉林敖东 | 80 | 0 | 80% |
| 比亚迪 | 80 | -5(竞争) | 75% |
| 小米集团 | 80 | -5(竞争) | 75% |
| 理想汽车 | 80 | -10(竞争 + 投入) | 70% |
| 三六零 | 80 | -15(转型不确定) | 65% |
| 大麦娱乐 | 50 | -5(持续亏损) | 50% |
| 地平线机器人 | 50 | -5(技术风险) | 45% |

## 六、Excel 模板使用指南

### 步骤 1：导入数据
```
1. 打开 Excel
2. 数据 → 从文本/CSV → 选择 valuation_data_20260314.csv
3. 选择"逗号"分隔 → 加载
```

### 步骤 2：添加公式列
```
在现有数据后添加以下列：

P 列 (安全边际)：=D2/N2*100
Q 列 (估值状态)：=IF(O2<60,"深度价值",IF(O2<80,"价值区间",IF(O2<95,"合理偏低",IF(O2<=105,"合理",IF(O2<=120,"合理偏高","高估")))))
R 列 (建议)：=IF(O2<50,"强烈买入",IF(O2<60,"买入",IF(O2<80,"增持",IF(O2<=105,"持有",IF(O2<=120,"减持","卖出")))))
```

### 步骤 3：条件格式
```
1. 选中安全边际列
2. 开始 → 条件格式 → 色阶
3. 选择"红 - 黄 - 绿"
4. 最小值=50, 中间值=100, 最大值=150
```

### 步骤 4：创建仪表板
```
1. 插入 → 数据透视表
2. 行：估值状态
3. 值：计数 (股票名称)
4. 插入 → 饼图
```

## 七、更新维护指南

### 定期更新频率：
| 数据类型 | 频率 | 负责工具 |
|----------|------|----------|
| 股价 | 每日收盘后 | 东方财富 API |
| PE/PB | 每日 | 自动计算 |
| EPS/BVPS | 季度财报后 | 手动更新 |
| ROE | 季度财报后 | 手动更新 |
| 内在价值 | 月度 | 重新计算 |
| 安全边际 | 每日 | 自动计算 |

### 触发重新估值的事件：
- 季度财报发布
- 重大并购/分拆
- 行业政策重大变化
- 股价波动超过±20%
- 公司指引大幅调整
