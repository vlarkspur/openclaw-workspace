| 编号 | 偏差名称 | 英文名称 | 典型表现 |
|------|----------|----------|----------|
| 1 | 过度自信 | Overconfidence | "我觉得肯定会涨" |

| 9 | 框架效应 | Framing | 同一信息不同表述影响决策 |

## 触发条件与应对策略

### 🚨 高危场景

| 场景 | 触发偏差 | 警告语 | 应对 |
|------|----------|--------|------|
| "我觉得会涨/跌" | 过度自信、控制幻觉 | "请提供数据支撑" | 要求 3 个独立数据源 |
| 想加仓但已超仓位 | 过度自信、FOMO | "提醒能力圈和风险" | 检查仓位上限 |
| 亏损后想"扳本" | 损失厌恶、赌徒谬误 | "警告报复性交易" | 强制冷静期 24h |
| FOMO 想追高 | 从众心理、可得性偏差 | "提醒安全边际" | 计算估值分位数 |
| 死扛不止损 | 损失厌恶、沉没成本 | "设定止损线" | 执行预设止损 |
| 过早止盈 | 处置效应、心理账户 | "检查卖出逻辑" | 用同一标准评估 |

## 检查清单（决策前必用）

### 📋 买入前检查清单

- [ ] 我有几个数据源支撑这个判断？（至少 3 个）
- [ ] 我是否在 FOMO？（检查情绪状态）
- [ ] 仓位是否超过上限？（单只≤20%，总仓≤100%）
- [ ] 我是否只看了利好消息？（强制找 3 个利空）
-  [ ] 估值处于什么分位数？（历史百分位）
- [ ] 最坏情况是什么？（能承受吗）
- [ ] 我是否因为"熟悉"而买？（检查熟悉度偏差）
- [ ] 反过来想：什么情况下会亏钱？

### 📋 卖出前检查清单

- [ ] 卖出理由与买入理由一致吗？
- [ ] 我是否因为亏损而不想卖？（损失厌恶）
- [ ] 我是否因为盈利而想卖？（处置效应）
- [ ] 估值是否真的过高？（数据支撑）
- [ ] 基本面是否恶化？（客观指标）
- [ ] 是否有更好的投资标的？（机会成本）
- [ ] 反过来想：什么情况下卖飞会后悔？

### 📋 持仓检查清单（周度）

- [ ] 每只股票的仓位占比是否正常？
- [ ] 是否有单只股票超过 20%？
- [ ] 是否有股票触及止损线？
- [ ] 最近是否有情绪化操作？
- [ ] 是否记录了每次决策的理由？

## 历史案例库（芒格智慧）

### 1. 损失厌恶案例
**场景：** 投资者死扛亏损股，不愿止损  
**芒格点评：** "如果发现自己身处坑中，最重要的事情就是停止挖掘。"  
**应对：** 预设止损线，机械执行

### 2. 过度自信案例
**场景：** 投资者认为能预测市场走势  
**芒格点评：** "预测不能告诉你什么，但它可以告诉你很多关于预测者的信息。"  
**应对：** 要求数据支撑，计算胜率

### 3. 从众心理案例
**场景：** FOMO 追高热门股  
**芒格点评：** "别人贪婪时我恐惧，别人恐惧时我贪婪。"  
**应对：** 检查估值分位数，计算安全边际

### 4. 确认偏误案例
**场景：** 只看利好消息，忽略利空  
**芒格点评：** "如果我要证明某个观点，我会先找反面证据。"  
**应对：** 强制找 3 个利空理由
